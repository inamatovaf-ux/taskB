sin x = a cos x = a tg x = a ctg x = a Примеры Задания ✏️
Главная идея
Единичный круг — это координатная плоскость
Что обозначает каждая функция
sin(α) — координата по оси Y (высота точки)
cos(α) — координата по оси X (ширина точки)
tg(α) — отрезок на касательной x = 1
ctg(α) — отрезок на касательной y = 1
Алгоритм решения (без зубрёжки!)
1
Рисуй единичный круг с осями X и Y
2
Найди нужное число на нужной оси (Y для sin, X для cos)
3
Проведи прямую через эту точку (горизонтальную для sin, вертикальную для cos)
4
Прочитай углы двух точек пересечения с кругом — это ответы
5
Добавь +2πn (полные обороты) — получишь все решения
sin x = a
Синус: смотрим на ось Y
Ищем число a на оси Y → горизонтальная линия → две точки на круге
30°
sin(α)
0.500
Два ответа
π/6 ; 5π/6
1
Рисуем круг с осями. На оси Y отмечаем значения: −1, −0.5, 0, 0.5, 1.
2
Находим значение 0.5 на оси Y. Идём по вертикальной оси вверх до отметки 0.5 и ставим там точку.
3
Чертим горизонтальную линию y = 0.5 — из этой точки влево и вправо, через весь круг. Она пересечёт круг в двух точках.
4
Отмечаем обе точки и читаем углы:
• Правая точка (I четверть): α₁ = 30° = π/6
• Левая точка (II четверть): α₂ = 150° = 5π/6
5
Записываем ответ (+2πn):
x = π/6 + 2πn  или  x = 5π/6 + 2πn, n ∈ ℤ
❓ Почему именно 5π/6? Откуда берётся второй угол?
π/6 = 30° — первый угол, синус которого равен 0.5 (стандартный угол).
5π/6 = 150° — вторая точка пересечения линии y = 0.5 с кругом. Она симметрична первой относительно оси Y.

📐 Правило второго угла для синуса: второй угол = π − первый
π − π/6 = 6π/6 − π/6 = 5π/6 ✓
sin x = a  →  x = π/6 + 2πn  или  x = 5π/6 + 2πn
В общем виде: x = (−1)ⁿ arcsin(a) + πn, n ∈ ℤ
cos x = a
Косинус: смотрим на ось X
Ищем число a на оси X → вертикальная линия → две точки на круге
60°
cos(α)
0.500
Два ответа
±π/3+2πn
1
Рисуем круг с осями. На оси X отмечаем значения: −1, −0.5, 0, 0.5, 1.
2
Находим значение 0.5 на оси X. Идём по горизонтальной оси вправо до отметки 0.5.
3
Чертим вертикальную линию x = 0.5 — из этой точки вверх и вниз, через весь круг. Два пересечения с кругом — одно выше оси X, одно ниже.
4
Читаем оба угла:
• Верхняя точка (I четверть): α₁ = 60° = π/3
• Нижняя точка (IV четверть): α₂ = −60° = −π/3
5
Записываем ответ: углы симметричны → x = ±π/3 + 2πn
❓ Почему ±π/3, а не два отдельных угла как у синуса?
У синуса: π/6 и 5π/6 — они не равны по модулю, поэтому пишем раздельно.
У косинуса: π/3 и −π/3 — они равны по модулю, симметричны по оси X → пишем ±π/3.

📐 Правило второго угла для косинуса: второй угол = −первый
cos x = a  →  x = ±arccos(a) + 2πn
Знак ± = два угла: +arccos(a) и −arccos(a). Оба дают одинаковый косинус.
tg x = a
Тангенс: касательная x = 1
Откладываем a на касательной справа → луч из O → один угол → период π
45°
tg(α)
1.000
Ответ
π/4+πn
1
Рисуем касательную x = 1. Вертикальная прямая, которая касается круга в точке (1, 0) справа.
2
На касательной x = 1 откладываем значение a. Для tg x = 1: отмечаем высоту 1 → точка (1, 1).
3
Проводим луч из центра O через точку (1, 1). Угол этого луча с осью X — и есть ответ.
4
Читаем угол: α = 45° = π/4. Один класс решений с периодом π:
x = π/4 + πn, n ∈ ℤ
❓ Почему один ответ и период π, а не два ответа и 2π?
Для sin/cos — линия пересекает круг в двух разных точках → два угла → 2πn.
Для тангенса — один луч из O. Через 180° (π) луч смотрит в ту же сторону пересечения с касательной → период вдвое меньше!

📐 Проверка: tg(45°) = 1, tg(45° + 180°) = tg(225°) = 1 ✓
tg x = a  →  x = arctg(a) + πn
Один класс! Период π. Не определён при x = π/2 + πn.
ctg x = a
Котангенс: касательная y = 1
Как тангенс, но касательная горизонтальная (сверху круга)
60°
ctg(α)
0.577
Ответ
π/6+πn
1
Рисуем касательную y = 1. Горизонтальная прямая, касающаяся круга сверху в точке (0, 1).
2
Откладываем a по горизонтали на y = 1. Для ctg x = 1: точка (1, 1).
3
Луч из O через (1, 1) образует угол 45° = π/4.
x = π/4 + πn, n ∈ ℤ
❓ Разница между tg и ctg на круге
tg = sin/cos → касательная x = 1 (вертикальная, справа)
ctg = cos/sin → касательная y = 1 (горизонтальная, сверху)

💡 ctg(α) = 1/tg(α) — они взаимно обратные!
ctg x = a  →  x = arcctg(a) + πn
Один класс, период π. Не определён при x = πn (sin = 0).
