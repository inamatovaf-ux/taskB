Решаем через единичный круг — без зазубривания формул.
Главное — понять логику один раз.
sin x = a cos x = a tg x = a ctg x = a Примеры Задания ✏️
Главная идея
Единичный круг — это координатная плоскость
Что обозначает каждая функция
sin(α) — координата по оси Y (высота точки)
cos(α) — координата по оси X (положение по горизонтали)
tg(α) — отрезок на касательной x = 1 (вертикальная)
ctg(α) — отрезок на касательной y = 1 (горизонтальная)
Алгоритм решения (без зубрёжки!)
1
Нарисуй единичный круг с осями X и Y
2
Найди нужное число на нужной оси — Y для sin, X для cos
3
Проведи прямую через эту точку (горизонтальную для sin, вертикальную для cos)
4
Прочитай углы двух точек пересечения с кругом — это оба ответа
5
Добавь +2πn — получишь все бесконечные решения
sin x = a
Синус: ищем на оси Y
Находим a на оси Y → горизонтальная линия → ДВЕ точки на круге
30°
sin(α)
0.500
Два угла
π/6 ; 5π/6
1
Рисуем круг с осями. На оси Y отмечаем значения: −1, −0.5, 0, 0.5, 1.
2
Находим 0.5 на оси Y. Идём вверх по оси Y и ставим точку на отметке 0.5. Это наш ориентир.
3
Чертим горизонтальную линию y = 0.5 — влево и вправо через весь круг. Она пересечёт круг ровно в двух точках.
4
Отмечаем ОБЕ точки и читаем углы:
• Правая точка (I четверть): α₁ = 30° = π/6
• Левая точка (II четверть): α₂ = 150° = 5π/6
5
Записываем ответ:
x = π/6 + 2πn  или  x = 5π/6 + 2πn,  n ∈ ℤ
❓ Почему π/6 и 5π/6? Откуда второй угол?
π/6 = 30° — первый угол, sin которого = 0.5 (стандартный угол).
5π/6 = 150° — вторая точка пересечения линии y = 0.5. Она симметрична первой относительно оси Y.

📐 Правило: второй угол = π − первый угол
π − π/6 = 6π/6 − π/6 = 5π/6 ✓
sin x = a → x = π/6 + 2πn  или  x = 5π/6 + 2πn
В общем виде: x = (−1)ⁿ arcsin(a) + πn, n ∈ ℤ
cos x = a
Косинус: ищем на оси X
Находим a на оси X → вертикальная линия → ДВЕ точки на круге
60°
cos(α)
0.500
Два угла
±π/3+2πn
1
Рисуем круг с осями. На оси X отмечаем: −1, −0.5, 0, 0.5, 1.
2
Находим 0.5 на оси X. Идём вправо по горизонтальной оси и ставим точку на 0.5.
3
Чертим вертикальную линию x = 0.5 — вверх и вниз через весь круг. Два пересечения — одно выше оси X, другое ниже.
4
Читаем ОБА угла:
• Верхняя точка (I четверть): α₁ = 60° = π/3
• Нижняя точка (IV четверть): α₂ = −60° = −π/3
5
Углы симметричны по оси X → пишем компактно: x = ±π/3 + 2πn
❓ Почему ±π/3, а не два разных угла как у синуса?
У синуса: π/6 и 5π/6 — не равны по модулю → пишем раздельно.
У косинуса: π/3 и −π/3 — равны по модулю, симметричны по оси X → пишем ±π/3.

📐 Правило: второй угол = −первый угол
cos x = a → x = ±arccos(a) + 2πn
± означает два угла: +arccos(a) и −arccos(a). Оба дают одинаковый косинус.
tg x = a
Тангенс: касательная x = 1
Откладываем a на касательной справа → луч из O → один угол → период π
45°
tg(α)
1.000
Ответ
π/4+πn
1
Рисуем вертикальную касательную x = 1. Она касается круга в точке (1, 0) справа.
2
На x = 1 откладываем значение a. Для tg x = 1: высота 1 → точка (1, 1).
3
Луч из O через точку (1, 1). Он пересечёт окружность. Угол луча с осью X — это и есть ответ.
4
Угол: 45° = π/4. Период π (не 2π!)
x = π/4 + πn, n ∈ ℤ
❓ Почему один ответ и период π, а не 2π?
У sin/cos — линия даёт 2 разные точки → 2πn.
У tg — один луч из O. Через 180° (π) луч снова пересекает касательную x = 1 в той же точке → период вдвое меньше!

📐 Проверка: tg(45°) = 1, tg(225°) = 1 ✓
tg x = a → x = arctg(a) + πn
Один класс! Период π. Не определён при x = π/2 + πn.
ctg x = a
Котангенс: касательная y = 1
Как тангенс, но касательная горизонтальная (сверху)
60°
ctg(α)
0.577
Ответ
π/6+πn
1
Горизонтальная касательная y = 1 — касается круга сверху в точке (0, 1).
2
Для ctg x = 1 откладываем 1 по горизонтали → точка (1, 1).
3
Луч из O через (1, 1) → угол 45° = π/4.
x = π/4 + πn, n ∈ ℤ
❓ Разница между tg и ctg
tg = sin/cos → касательная x = 1 (вертикальная, справа)
ctg = cos/sin → касательная y = 1 (горизонтальная, сверху)

💡 ctg(α) = 1 / tg(α) — взаимно обратные!
ctg x = a → x = arcctg(a) + πn
Один класс, период π. Не определён при x = πn (sin = 0).
